• 1.40d, 9 April 2008
    • Converted the FluorescentCells sample image into a 3 channel composite image.
    • The Image Calculator now creates a hyperstack when the input is a hyperstack.
    • Fixed a bug that could cause macros not running in batch mode that are using the particle anlyzer on a stack to not run reliably.
    • Fixed a bug that caused reading of LZW compressed TIFF stacks to fail.
    • Fixed a bug that caused the LSM Reader to not correctly open LSM files using LZW with differencing compression.
    • Fixed a bug that caused Image>Color>RGB Merge, when creating a composite image, to sometimes not use expected channel colors.
