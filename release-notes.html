• 1.41e, 10 June 2008
    • Michael Schmid added the ability to save using Motion JPEG and PNG compression to the File>Save As>AVI command and added the ability to read JPEG and PNG compressed AVIs to the File>Open command.
    • Added the Macros>Evaluate JavaScript command to the macro editor. This command requires Java 1.6, which has the Mozilla Rhino JavaScript interpreter built in (example).
    • Added the Help>Update ImageJ command, which duplicates the Plugins>Utilities>Update ImageJ command.
    • Jerome Mutterer contributed the macro editor's new Macros>Find Functions command.
    • Edit>Selection>Rotate works with composite selections.
    • Edit>Selection>Restore Selection works with non-rectangular selections that extend beyond the boundary of the destination image.
    • The getStatistics() and getRawStatistics() macro functions now work with line selections.
    • The File.delete() macro function now returns "1" (true) if the file or directory was successfully deleted. The "1" will be displayed in the Log window if File.delete() is not called in an assignment or 'if' statement.
    • Single .class plugins in a directory are not moved to the MIscellaneous menu if there is a least one underscore in the directory name.
    • Fixed a bug that sometimes caused 16-bit signed stacks to not be correctly converted to 32-bits.
