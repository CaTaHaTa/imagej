• 1.39t, 3 March 2008
    • The histogram in the Image>Adjust>Brightness/Contrast window is now always updated when the B&C window is activated.
    • Added the "Are the lanes really horizontal?" dialog to the Analyze>Gels>Select First Lane command.
    • An error message is displayed when a calibrated is opened and "Global" calibration is enabled.
    • PNGs are again opened using ImageIO, which fixes several bugs (introduced in v1.39i) that caused 16-bit and RGB PNGs to not open or to not open correctly.
    • Added the getInfo("image.description") macro function.
    • Added the ImportDialog.getLastFileInfo() method.
    • Fixed a bug that caused File>Import>URL to fail to open GIF, JPEG, PNG and DICOM files with uppercase extensions.
    • Fixed several bugs in Edit>Selection>Specify, thanks to patches contributed by Joachim Wesner.
    • Fixed a bug in Image>Scale that could cause a blank result if the source was an RGB image. the x scale was less than 0.25 and the y scale was greater than 0.5.
